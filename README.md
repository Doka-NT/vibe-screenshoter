# Vibe Screenshoter

Простое приложение для macOS, которое работает в строке меню и позволяет делать скриншоты.

## Особенности

- Работает исключительно в строке меню (menu bar)
- Иконка камеры (SF Symbol `camera.viewfinder`) для индикации функции скриншотера
- Минималистичный интерфейс
- Адаптируется к светлой и темной теме macOS
- Не отображается в Dock благодаря `LSUIElement = YES`

## Требования

- macOS 13.0 или выше
- Xcode 15.0 или выше
- Swift 5.0 или выше

## Установка и запуск

### Открытие проекта

1. Откройте `VibeScreenshoter.xcodeproj` в Xcode
2. Выберите схему VibeScreenshoter
3. Нажмите ⌘R для запуска приложения

### Сборка приложения

В Xcode выберите Product → Archive для создания релизной версии приложения.

## Функциональность

Приложение создает иконку в строке меню macOS с следующими опциями:

- **Сделать снимок экрана** (⌘S) - захват всего экрана
- **Сделать снимок области** (⌘A) - захват выбранной области
- **О приложении** - информация о версии
- **Выход** (⌘Q) - закрытие приложения

## Технические детали

### Иконка в меню баре

Приложение использует SF Symbol `camera.viewfinder` как иконку в строке меню. Иконка автоматически адаптируется к светлой и темной темам благодаря свойству `isTemplate = true`.

### Режим menu bar-only

Приложение настроено как агент (LSUIElement = YES в Info.plist), что означает:
- Не отображается в Dock
- Не имеет главного окна
- Работает только через иконку в строке меню
- Не переключает фокус при запуске

### AppDelegate

Класс `AppDelegate` управляет жизненным циклом приложения и создает:
- Status bar item (иконку в строке меню)
- Контекстное меню с опциями
- Обработчики действий для каждого пункта меню

## Разработка

Базовая структура приложения готова. Функции захвата экрана требуют дальнейшей реализации:
- Интеграция с системным API для захвата экрана
- Сохранение скриншотов в файловую систему
- Выбор области экрана для захвата
- Настройки приложения (формат файлов, путь сохранения)
