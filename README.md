# Vibe Screenshoter

Легковесное приложение для создания скриншотов на macOS с возможностью добавления разметки.

## Функциональные возможности

### Основные функции

1. **Запуск и Иконка в Меню Баре**
   - Программа автоматически запускается при входе в систему (опционально)
   - Отображается в меню баре в виде иконки
   - Контекстное меню с опциями:
     - Запустить скриншот
     - Настройки
     - Выход из программы

2. **Горячие клавиши**
   - По умолчанию: `⌘⇧S` (Command + Shift + S)
   - Настраиваемая комбинация клавиш

3. **Создание скриншота**
   - Выбор произвольной области экрана с помощью мыши
   - Отмена по нажатию `ESC` или правой кнопки мыши
   - Отображение размеров выделенной области

4. **Редактор скриншота**
   - **Инструменты разметки:**
     - Стрелка
     - Текст
     - Прямоугольник
   - **Настройка элементов:**
     - Изменение цвета
     - Изменение размера линий и текста
   - **Удаление элементов:**
     - Инструмент удаления для отдельных элементов разметки

5. **Сохранение**
   - Автоматическое копирование в буфер обмена
   - Сохранение в файловую систему
   - Настраиваемая папка для сохранения
   - Поддержка форматов: PNG (по умолчанию), JPEG
   - Автоматическое именование файлов с временной меткой

## Требования

- macOS 13.0 или выше
- Xcode 15.0 или выше (для сборки)
- Swift 5.9 или выше

## Сборка

### С помощью Swift Package Manager

```bash
# Клонируйте репозиторий
git clone https://github.com/Doka-NT/vibe-screenshoter.git
cd vibe-screenshoter

# Соберите проект
swift build -c release

# Запустите приложение
.build/release/VibeScreenshoter
```

### С помощью Xcode

1. Откройте `Package.swift` в Xcode
2. Выберите схему "VibeScreenshoter"
3. Нажмите `⌘R` для запуска или `⌘B` для сборки

## Использование

### Запуск

После запуска приложение появится в меню баре в виде иконки камеры.

### Создание скриншота

1. **С помощью горячих клавиш:**
   - Нажмите `⌘⇧S` (или настроенную комбинацию)

2. **Через меню:**
   - Кликните на иконку в меню баре
   - Выберите "Запустить скриншот"

3. **Выбор области:**
   - Зажмите левую кнопку мыши
   - Выделите нужную область
   - Отпустите кнопку мыши

### Редактирование

После выбора области откроется редактор:

1. **Добавление стрелки:**
   - Нажмите кнопку "Стрелка"
   - Нарисуйте стрелку от начальной до конечной точки

2. **Добавление текста:**
   - Нажмите кнопку "Текст"
   - Кликните в нужном месте
   - Введите текст в диалоговом окне

3. **Добавление прямоугольника:**
   - Нажмите кнопку "Прямоугольник"
   - Нарисуйте прямоугольник

4. **Удаление элемента:**
   - Нажмите кнопку "Удалить"
   - Кликните на элемент, который нужно удалить

5. **Настройка:**
   - Используйте цветовой пикер для выбора цвета
   - Используйте ползунок для изменения толщины линий

6. **Сохранение:**
   - Нажмите кнопку "Сохранить"
   - Скриншот будет скопирован в буфер обмена и сохранен в файл

### Настройки

Откройте настройки через меню бар:

- **Запуск при входе:** Автоматический запуск при входе в систему
- **Горячая клавиша:** Настройка комбинации клавиш (в разработке)
- **Путь для сохранения:** Выбор папки для сохранения скриншотов
- **Формат сохранения:** PNG или JPEG

## Архитектура

Проект состоит из следующих компонентов:

- **AppDelegate**: Главный делегат приложения, управляет жизненным циклом
- **ScreenshotManager**: Управление процессом создания скриншотов
- **CaptureWindow**: Окно для выбора области экрана
- **EditorWindow**: Редактор для добавления разметки
- **SettingsManager**: Управление настройками приложения
- **SettingsWindowController**: Окно настроек
